
# FlowFocus 🌿

## Overview
FlowFocus is an AI-powered productivity chatbot designed to help users improve focus, manage time effectively, build healthy habits, and overcome procrastination through interactive guidance and structured planning.

## Features
- Interactive chatbot-based productivity guidance
- Personalized focus and habit-building suggestions
- Time management and goal-setting assistance
- Morning routine and sleep improvement tips
- Deep work and study system recommendations
- Smooth animated UI with modern responsive design
- Dark mode support

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
The main objective of FlowFocus is to provide students and users with a simple digital assistant that promotes productivity, discipline, and self-improvement in everyday life.

## Future Enhancements
- Progress tracking and streak monitoring
- Save daily productivity plans
- AI-based personalized recommendations
- Database integration for user history
